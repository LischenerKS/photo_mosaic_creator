# Photo Mosaic Creator
Создает изображение, каждый пиксель которого заменен на другое изображение, и открывает его в стандартном приложении для просмотра фото.

## Содержание
- [Использование](#Использование)
- [Требования](#Требования)
- [Установка зависимостей](#Установка-зависимостей)

## Использование
Склонируйте репозиторий и перейдите в полученную директорию

```sh
$ git clone https://github.com/LischenerKS/photo_mosaic_creator.git
$ cd ./photo_mosaic_creator/
```
Запустите main.py
```sh
$ python main.py
```

## Требования
Для установки и запуска проекта, необходим [Python](https://www.python.org/downloads/) v3.14+

## Установка зависимостей
Для установки зависимостей, выполните команду:
```sh
$ pip install Pillow
```
